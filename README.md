# Username Sniper

A fast and lightweight Discord 4-digit username availability checker.

---

## ⚡ Features

### 🎯 Username Sniper

Looking for 4 digit discord usernames

---

## 🧠 Requirements

Before using the tool, you **MUST** install Python.

### 📌 Install Python

https://www.python.org/downloads/

⚠️ During installation make sure to enable:

- Add Python to PATH

---

## 📦 Installation

### Option 1

1. Download the project.
2. Extract the folder.
3. Run:

```bat
setup.bat
```

This will:

- Check your Python installation
- Install all required dependencies
- Launch the application automatically

---

### Option 2 (Manual)

Install the required packages:

```bash
pip install -r requirements.txt
```

Then start the tool:

```bash
python main.py
```

---

## 📥 Dependencies

- colorama
- requests
- pywin32
- pycryptodome

---

## ▶️ Usage

After running **setup.bat**, the tool starts automatically.

Or launch it manually:

```bash
python main.py
```
